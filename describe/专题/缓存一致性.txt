计算机中遇到的并发问题：
由于计算机的内存与CPU运算速度有几个数量级的差别，所以电脑中加入了尽可能接近CPU运算速度的高速缓存：
运算过程：将运算需要用到的数据复制到缓存中，让运算快速运行，当运算结束后，再从缓存中同步回内存中
缓存解决了CPU与内存的矛盾，但是引入了一个更加复杂的问题：缓存一致性问题

在多处理器系统中，每个处理器有自己的缓存，而它们有共享同一主存

为了使得处理器内部的运算单元尽量被充分利用，CPU会对输入的代码乱序执行，out of order execution

JMM java memory model Java内存模型

所有的变量都存储在主内存中，每个线程有自己的工作内存，对变量的操作必须在工作内存中,线程之间的变量必须通过主内存来完成
线程对变量的操作必须在工作中完成
